A chat app built on sails.js and node.js
 
Installation:

From terminal-

-> cd into the directory you'd like to install the app.

-> Run "git clone git@github.com:Ajeey/node-sails-chat.git"

-> cd into the newly created directory.

-> Run "npm install".

-> Run "sails lift".
